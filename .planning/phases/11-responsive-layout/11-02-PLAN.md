**1. Task runs card markup (list-rows.eta):**

Wrap the existing `<table>` elements in a `<div class="desktop-only">` wrapper (both the empty-state table and the populated table).

After each table wrapper, add a mobile card list. For the empty state:
```html
<div class="mobile-only" style="text-align: center; color: #64748B; padding: 2rem;">No tasks found.</div>
```

For the populated state, add a `<div class="task-card-list mobile-only">` containing one `.task-card-wrapper` per run. Each wrapper contains:
- A `.swipe-action-approve` div (hidden by default, used by Plan 03 swipe gestures) — only rendered if `run.status === 'draft_ready'`
- A `.task-card` div with `data-task-id="<%= run.id %>"` and `data-status="<%= run.status %>"` containing:
  - `.task-card-header`: client name (left) + status badge (right, using existing `.badge.badge-<status>` classes)
  - `.task-card-meta`: channel badge + task type text
  - `.task-card-footer`: formatted date (left) + created_by (right)
- The card is clickable: `onclick="window.location='/tasks/<%= run.id %>'"` on the `.task-card` element

The `.swipe-action-approve` structure (to be wired by Plan 03):
```html
<div class="swipe-action-approve" style="display: none;">
  <svg width="20" height="20" viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="2.5"><polyline points="20 6 9 17 4 12"/></svg>
  <span>Approve</span>
</div>
```

**2. Client card markup (list-table.eta):**

Wrap the existing `<table>` in a `<div class="desktop-only">` wrapper.

After it, add a `<div class="client-card-list mobile-only">` containing one `.client-card` per client:
- Each card is an `<a>` tag linking to `/clients/<%= encodeURIComponent(c.name) %>`
- `.client-card-row` (top): client label (left) + health dot (right, same colour logic as the table — green >= 70, amber >= 40, red < 40, grey if null)
- `.client-card-row` (bottom, smaller text): vertical on left, health score on right (admin only)
- For the empty state: a `.mobile-only` div with "No clients match the current filters."
