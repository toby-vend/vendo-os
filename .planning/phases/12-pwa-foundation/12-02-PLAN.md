  <action>
Append a new card after the existing "Connected Accounts" card (before the closing `</div>` of the max-width container) in `web/views/settings.eta`. The card contains an "Install App" section with an inline script that detects the user's platform and renders the appropriate content.

The card markup and script (per user decision and RESEARCH.md Pattern 5):

```html
<!-- Install App -->
<div class="card" style="padding: 1.5rem; margin-top: 1.5rem;" id="install-app-section">
  <h3 style="font-size: 16px; font-weight: 600; margin-bottom: 1rem;">Install App</h3>
  <div id="install-content"></div>
</div>

<script>
(function() {
  var el = document.getElementById('install-content');
  if (!el) return;

  // Already installed — standalone mode detection
  if (window.matchMedia('(display-mode: standalone)').matches || window.navigator.standalone) {
    el.innerHTML = '<p style="font-size:14px;color:#22C55E;">App installed &#10003;</p>';
    return;
  }

  var isIOS = /iphone|ipad|ipod/i.test(navigator.userAgent);

  if (isIOS) {
    el.innerHTML = '<p style="font-size:14px;color:#E2E8F0;line-height:1.6;">'
      + 'To install VendoOS on your home screen:<br>'
      + '1. Tap the <strong>Share</strong> button (box with arrow) in Safari<br>'
      + '2. Scroll down and tap <strong>Add to Home Screen</strong><br>'
      + '3. Tap <strong>Add</strong> to confirm</p>';
    return;
  }

  // Android / Chrome — use captured beforeinstallprompt if available
  if (window._pwaInstallPrompt) {
    el.innerHTML = '<button id="pwa-install-btn" style="padding:8px 20px;font-size:14px;'
      + 'color:#0B0B0B;background:#22C55E;border:none;border-radius:8px;cursor:pointer;'
      + 'font-family:var(--vendo-font);font-weight:600;">Add to Home Screen</button>';
    document.getElementById('pwa-install-btn').addEventListener('click', function() {
      window._pwaInstallPrompt.prompt();
    });
    return;
  }

  // Generic fallback (desktop or Android without captured prompt)
  el.innerHTML = '<p style="font-size:14px;color:#94A3B8;">Open VendoOS in Chrome on your phone '
    + 'and use the browser menu to select <strong>Add to Home Screen</strong>.</p>';
})();
</script>
```

Key implementation notes:
- Use `&#10003;` for the checkmark character (not an emoji — per CLAUDE.md convention).
- The `window._pwaInstallPrompt` is set by the beforeinstallprompt listener added to base.eta in Plan 01.
- The `isIOS` check uses UA string; `beforeinstallprompt` does NOT exist on iOS Safari — the iOS branch must show manual instructions, never a button.
- The section always renders server-side (no SSR platform detection needed); the inline script populates the content client-side.
- The install section remains permanently accessible in Settings (per user decision: "If user dismisses or ignores, section remains in Settings").
