Draw a green "V" character (Manrope 700, colour #22C55E) centred on a dark (#0B0B0B) background with rounded corners (border-radius ~12% of canvas). Generate:
   - `public/assets/icon-192.png` (192x192, "V" fills ~65% of canvas)
   - `public/assets/icon-512.png` (512x512, "V" fills ~65% of canvas)
   - `public/assets/icon-maskable-192.png` (192x192, "V" within inner 80% safe zone — 19px padding each side)
   - `public/assets/icon-maskable-512.png` (512x512, "V" within inner 80% safe zone — 51px padding each side)
   Since sharp cannot render text natively, create the icons using an SVG-to-PNG approach: build an SVG string with the "V" character (using Manrope or a sans-serif fallback — the font may not be available system-side, so use a bold sans-serif as fallback), then convert via `sharp(Buffer.from(svgString)).png().toFile(...)`. Run the script after creating it and commit the output PNGs.

3. Create `public/sw.js`:
   ```javascript
   importScripts('https://storage.googleapis.com/workbox-cdn/releases/7.4.0/workbox-sw.js');

   workbox.core.skipWaiting();
   workbox.core.clientsClaim();

   // Static assets — CacheFirst with 30-day expiry
   workbox.routing.registerRoute(
     ({ request }) =>
       request.destination === 'style' ||
       request.destination === 'script' ||
       request.destination === 'image' ||
       request.destination === 'font',
     new workbox.strategies.CacheFirst({
       cacheName: 'vendo-static-v1',
       plugins: [
         new workbox.expiration.ExpirationPlugin({
           maxEntries: 60,
           maxAgeSeconds: 30 * 24 * 60 * 60,
         }),
       ],
     })
   );
   // Navigation and API requests pass through to network — Phase 13 adds offline handling
   ```

4. Update `vercel.json`: Add two route entries to the `routes` array, inserted AFTER the `/assets/(.*)` route and BEFORE the catch-all `/(.*) → /api/index.ts`:
   ```json
   { "src": "/sw.js", "headers": { "Cache-Control": "public, max-age=0, must-revalidate" }, "dest": "/public/sw.js" },
   { "src": "/manifest.json", "dest": "/public/manifest.json" }
   ```
   The legacy `"builds"` format does NOT support top-level `"headers"` — the Cache-Control MUST be an inline `"headers"` property on the route entry.

   // === Phase 12: PWA Service Worker & Install ===
   if ('serviceWorker' in navigator) {
     window.addEventListener('load', function() {
       navigator.serviceWorker.register('/sw.js')
         .catch(function(err) { console.warn('SW registration failed:', err); });
     });

   window.addEventListener('beforeinstallprompt', function(e) {
     e.preventDefault();
     window._pwaInstallPrompt = e;
   });
