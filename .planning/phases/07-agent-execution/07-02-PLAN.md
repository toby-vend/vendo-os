    Add a `generateDraft` function (not exported — called internally by assembleContext):

    ```typescript
    import Anthropic from '@anthropic-ai/sdk';
    import { loadTaskTypeConfig } from './task-types/index.js';
    import { updateTaskRunOutput } from './queries/task-runs.js';

    async function generateDraft(
      taskRunId: number,
      channel: string,
      taskType: string,
      skills: SkillSearchResult[],
      brandFiles: BrandHubRow[],
      clientName?: string,
    ): Promise<void> { ... }
    ```

    Implementation details:
    - Load config via `loadTaskTypeConfig(channel, taskType)`
    - Build sopContent: `skills.map(s => '### ' + s.title + '\n' + s.content.slice(0, 2000)).join('\n\n')` — truncate per-SOP at 2000 chars, log warning if truncated
    - Build brandContent: `brandFiles.map(b => '### ' + b.title + '\n' + b.content).join('\n\n')`
    - Get clientName from first brandFile's client_name (or 'Unknown' if no brand files)
    - Call `config.buildSystemPrompt(sopContent)` and `config.buildUserMessage(taskType, brandContent, clientName)`
    - Instantiate Anthropic client with `process.env.ANTHROPIC_API_KEY` — throw if missing
    - Call `client.messages.create()` with:
      - model: 'claude-sonnet-4-6'
      - max_tokens: 2048
      - system: systemPrompt
      - messages: [{ role: 'user', content: userMessage }]
      - output_config: { format: { type: 'json_schema', schema: config.schema } }
    - Parse response: `response.content.find(b => b.type === 'text')` then `JSON.parse(textBlock.text)`
    - Validate sources array is non-empty; if empty, throw to trigger retry
    - On success: `await updateTaskRunOutput(taskRunId, JSON.stringify(parsed))`
    - Retry logic: wrap API call in a loop (max 2 attempts). On first failure, wait 1 second then retry. On second failure, call `updateTaskRunStatus(taskRunId, 'failed')` and return (do not throw — this runs in fire-and-forget)
