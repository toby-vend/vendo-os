**2. DriveChange interface** (exported):
```typescript
export interface DriveChange {
  changeType: 'file' | 'drive';
  fileId: string;
  removed: boolean;
  file?: {
    id: string;
    name: string;
    mimeType: string;
    trashed: boolean;
    parents: string[];
    modifiedTime: string;
  };
}
```

**3. listChanges(userId: string, pageToken: string): Promise<{ changes: DriveChange[]; newPageToken: string }>**
- Paginate through `changes.list` API using the pattern from 03-RESEARCH.md.
- Request fields: `changes(changeType,fileId,removed,file(id,name,mimeType,trashed,parents,modifiedTime)),nextPageToken,newStartPageToken`.
- Include `includeRemoved: 'true'` and `spaces: 'drive'`.
- Handle pagination: loop while `nextPageToken` exists; return `newStartPageToken` as `newPageToken`.
- Use `getGoogleAccessToken(userId)` for auth.

**4. resolveChannel(fileId: string, accessToken: string): Promise<string | null>**
- Walk parent chain up to 5 levels using `files.get?fields=id,parents`.
- At each level, check if the folder ID is in `CHANNEL_FOLDER_MAP`.
- Return the channel slug or null if no match found.
- Drive files have exactly one parent (single-element `parents` array).

**5. extractContent(fileId: string, mimeType: string, accessToken: string): Promise<string | null>**
- `application/vnd.google-apps.document` → `files.export?mimeType=text/plain`
- `application/vnd.google-apps.spreadsheet` → return null (metadata only per user decision)
- `text/plain` or `text/markdown` → `files.get?alt=media`
- All other MIME types → return null
- Wrap in try/catch — on 403 (export size limit), log warning and return null.

**6. resolveSkillType(fileId: string, accessToken: string): Promise<string>**
- Get file's parent via `files.get?fields=id,parents`.
- If parent is a top-level channel folder (in CHANNEL_FOLDER_MAP), return `'sop'`.
- Otherwise, get parent's name via `files.get?fields=id,name`, slugify it: lowercase, replace non-alphanumeric with `_`, trim leading/trailing `_`.
- Return slugified name or `'sop'` as fallback.

**7. hashContent(content: string): string**
- `createHash('sha256').update(content, 'utf8').digest('hex')`
